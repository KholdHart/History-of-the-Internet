The World Wide Web is a platform created on top of the web, in other words, the internet came before the WWW. The internet is a global network infrastructure that connects millions of devices worldwide. It’s a network that enables data transfer and communication. The internet relies on various technologies, protocols, and services that enable seamless connectivity.

The World Wide Web is a platform created on top of the web, in other words, the internet came before the WWW. It consists of interconnected documents and resources (web pages) that are accessible through web browsers. The web uses a client-server model where web browsers (the clients) request and display web content stored on web servers.


A key differences between the two include:

  • The Internet uses TCP/IP protocols while WWW uses HTTP protocols.
  • The internet was created in the 60's and the WWW was created in 1989.
  • The WWW utilizes hyperlinks
