Overview Of The World Wide Web

The World Wide Web, otherwise known as WWW or W3, is a term that describes a system of interconnected documents and pages that are accessed with web browsers through the internet. The World Wide Web revolutionized communication, information sharing making it more accessible and efficient and this is done by allowing users to navigate using hyperlinks.
The web was created by British computer scientist Sir Tim Berners-Lee in 1989 who had the big intention of connecting everyone
The web's basis lies in its innovative concept of hypertext, where text contains links (known as hyperlinks), directing users to other relevant documents.

Some web browsers that allow users to use the W3 include

  • Google Chrome
  • Firefox
  • Opera
  • MS Edge
  • Internet Explorer
  • Bing
Over the years, the web has evolved with the development of new technologies and standards. These include CSS (Cascading Style Sheets) for styling web pages, JavaScript for adding interactivity, and more recently, HTML5 for multimedia integration and improved semantic structure.
